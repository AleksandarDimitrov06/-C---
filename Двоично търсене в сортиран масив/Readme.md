# Двоично търсене в сортиран масив

## Задача 
Напишете програма, която използва двоично търсене за намиране на число в сортиран масив от числа.

### Изисквания
* Двоичното тьрсене в сортиран масив оформете във функция BinSearchArrayRecursively,
която използва рекурсия намиране на число;
* Функцията BinSearchArrayRecursively трябва връща boolean, указващ дали числото е
намерено, и int - индекса на неговия пореден номер.

#### Допълнителни задачи
* Добавете функция ChecklfSorted, която проверява дали подаденият масив е сортиран, и
я използвайте. Ако масивът не е сортиран, изведете съобщение за грешка и не
извършвайте тьрсене;
* Добавете аналогична функция BinSearchArraylteratively, която осъществява двоично
търсене с използване на цикли.

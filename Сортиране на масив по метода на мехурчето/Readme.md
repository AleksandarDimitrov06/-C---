# Сортиране на масив по метода на мехурчето

## Задача

Създайте програма, в която се въвежда масив от дробни числа, масивът се сортира във възходящ ред по метода на мехурчето и след това се извеждат сортираните числа.

Допълнителни задачи
* Потребителят да може да избере дължината на въвеждания масив;
* Потребителят да може да избере между възходящ и низходящ ред на сортирането.
